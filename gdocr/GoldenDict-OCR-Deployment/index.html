  1. Package Release Notes「发行包说明」
  • macOS: Qt 5.9.9, x86_64, 10.13.6 or later「适用于 macOS 10.13.6 及以上系统, x86_64 平台」
  • macOS: Qt 5.15.17, x86_64/arm64, 10.13~15.6「适用于 macOS 10.13~15.6, x86_64/arm64 平台」

注意：从任何版本的GoldenDict及旧版GoldenDict++升级到新版时，在系统偏好设置安全性与隐私隐私页面中已有的与GoldenDict相关的设置将不再有效(同时运行多个版本的GD和GD++时相关隐私设置内容仅能对一个GD或GD++实例有效)，故在首次运行新版本前请务必先删除已存在的相关隐私允许项，在新版本首次运行时根据系统提示再逐一添加即可，依系统提示在添加相关的允许项后可能需要重新启动GoldenDict++才能生效。如果遇到无法打开App的相关问题，请参考苹果使用手册打开来自身份不明开发者的 Mac App，如果提示应用已损坏，请参考macOS 绕过公证和应用签名方法。

  • Linux: Qt 4.8.7, x86_64, Debian 13 or Compatible「适用于 Debian 13 x86_64 及与之兼容的 Linux 系统」

执行与GoldenDict文件同目录的同名(小写)sh脚本即可。针对特定语言(特别是小语种)内容的词典，如遇到豆腐块(不能识码显示)时，请安装相应语言的字体解决。

  • MinGW-w64: Qt 4.8.7, Windows XP sp3 or later「适用于 Windows XP sp3 及以上系统」
  • MinGW-w64: Qt 5.9.9, Windows 7 sp1 or later「适用于 Windows 7 sp1 及以上系统」
  • MinGW-w64: Qt 5.15.17, Windows 7 sp1 x64 or later「适用于 Windows 7 sp1 x64 及以上系统」
  • MSVC-2019: Qt 5.9.9, Windows 7 sp1 x64 or later「适用于 Windows 7 sp1 x64 及以上系统」

VC-Redist x64 or x86 is required by msvc2019 versions and should be installed first.「msvc2019版需先下载并安装运行时库x64或x86」

关于GoldenDict++的版本号：16进制方式表示，共四位，首位为MAJOR版本，次位为版本类型，第三位为MINOR版本，末位为PATCH版本。如，2A37实为A型的2.3.7版，2A3C实为A型的2.3.12版。具体以关于对话框中显示的内容为准。
