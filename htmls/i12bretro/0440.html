Running a CloudReady ChromeOS VM in Virtualbox 🌱

ðŸ’¡ NOTE: This method uses an older version of CloudReady Free. The latest release is incompatible with VirtualBox due to graphics requirements

  1. Open a web browser and download the CloudReady .ova Download
  2. Launch VirtualBox
  3. Select File > Import Appliance...
  4. Click the Browse icon > Navigate to the downloaded .ova file > Select CloudReady_Free_x64_Virtualbox.ova
  5. Click Next and then Import
  6. Make sure the CloudReady_Free_x64 VM is selected and click Start > Normal
  7. Select a Language and Keyboard Layout > Click Continue
  8. Click Accept and install
  9. Click Continue without Flash
  10. Enter your Google/GMail account and password
  11. Select an icon for your user account > Click OK
  12. Click the Take a tour button
  13. Welcome to CloudReady, ChromeOS
