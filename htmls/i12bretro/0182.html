Web Based Active Directory Management with ManageEngine ADManager Plus 🌱

  1. Download ADManager Plus Download
  2. Install ADManager Plus, skipping the registration step
  3. Once the installation completes, open a web browser and navigate to http://DNSorIP:8080
  4. Login with username admin and password admin
  5. Click the Click here to add a new domain link
  6. Complete the form, enabling the Authentication checkbox and entering the Domain Name, Username and Password
  7. Click the Discover New link
  8. Type the domain controller host name(s) > Click Add
  9. Click Add to complete setting up the domain
  10. Welcome to web based Active Directory administration with ADManager Plus
