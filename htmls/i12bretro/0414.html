Join VMware vSphere (ESXi) Host to Active Directory Domain 🌱

Create DNS entry

  1. Click the Start button > Windows Administrative Tools > DNS
  2. Expand the DNS server and domain name
  3. Right Click the domain name > New Host (A or AAAA)
  4. Type the dns name and target IP address > Click OK

Create ESX Admins Group

  1. Click the Start button > Windows Administrative Tools > Active Directory Users and Computers
  2. Expand the domain name
  3. Right click on the Users container > New > Group
  4. Name the group ESX Admins > Click OK
  5. Double click the newly created ESX Admins group
  6. Select the Members tab > Click the Add... button
  7. Add users who will be administrators for the ESXI host

Joining the Domain

  1. Open a web browser and navigate to the VMware vSphere web UI
  2. Log in with an administrative account
  3. Click Manage in the left navigation menu
  4. Click Time & date in the left sub navigation menu
  5. Select Use Network Time Protocol (enable NTP client)
  6. Set the Startup policy to Start and stop with host
  7. Enter the domain controller IP address(es) in the NTP server area
  8. Click the Save button
  9. Click the Actions button > Start to start the NTP server
  10. Click the Security & users tab in the main content area
  11. Click Authentication in the left sub navigation menu
  12. Click the Join Domain button
  13. Fill out the join domain fields providing a domain admin account
  14. Click the Join domain button
