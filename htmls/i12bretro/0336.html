Install Dig Frontend on Amazon FireTV Devices 🌱

  1. Install the Downloader application
  2. Launch Downloader
  3. Navigate to https://apkpure.com/dig-emulator-front-end/com.digdroid.alman.dig
  4. Click the Download APK button
  5. A prompt will display to go to settings and allow installing unknown apps
  6. Click Settings > Install Unknown Apps > Downloader > Toggle the setting to On
  7. Click back a few times to return to the Downloader app
  8. Click Install and then Install again
