Sideload Dig Emulation Frontend Onto Ouya Console 🌱

  1. On the Ouya, go to Manage > System > Advanced
  2. Scroll down and select Security
  3. Scroll down and check the Unknown sources box > Select OK to confirm
  4. Return to the home screen and select Make > Software > Browser
  5. Navigate to https://apkpure.com/dig-emulator-front-end/com.digdroid.alman.dig
  6. Click the Download APK button
  7. Select Browser to complete the download action
  8. Click back a few times to return to the Make > Software area
  9. Select an installed file browser application such as ES File Explorer
  10. Navigate to Download to find the downloaded Dig APK
  11. Select the .apk file and select Install
  12. Alternatively, if FilePwn is installed on the Ouya return to the home screen and select Play > FilePwn
  13. Navigate to Download to find the downloaded Dig APK
  14. Select the .apk file
  15. Select Package installer to complete the install action
  16. Click Install and wait for the installation to complete
