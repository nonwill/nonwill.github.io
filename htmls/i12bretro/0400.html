Update 4K FireTV Stick -- NEW REDESIGNED UI!! 🌱

  1. From the home screen, select Settings > My Fire TV > About
  2. Scroll to the bottom and select Check for Updates
  3. If the update is available it will begin downloading
  4. Once the download completes, click Install Update
  5. Wait for the update to be applied 
  6. The FireTV stick will reboot
  7. Welcome to the new, redesigned FireOS user interface
