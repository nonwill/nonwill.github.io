Install FTP Server on Amazon FireTV Devices 🌱

  1. Search for ES File Explorer
  2. Install ES File Explorer
  3. Launch ES File Explorer
  4. On the home menu, click the View on PC button
    • Alternatively, expand Network on left navigation > View on PC
  5. Click the big Turn On button to start the FTP server
  6. The IP address and port will display, by default anonymous access is enabled
  7. To set authentication click Settings below the Turn On button
  8. Select Set Manage Account
