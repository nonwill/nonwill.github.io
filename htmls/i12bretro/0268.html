Install OpenMediaVault on Armbian 🌱

  1. Log into the Armbian device
  2. Run the following commands in a terminal:
    # update software repositories and install available updates
    sudo apt update
    sudo apt upgrade
    # download and run OpenMediaVault installer
    wget -O - https://github.com/OpenMediaVault-Plugin-Developers/installScript/raw/master/install | sudo bash
  3. Open a web browser and navigate to http://localhost
  4. Log into OMV with username admin and password openmediavault
  5. Welcome to OpenMediaVault running on a cheap single board computer

Guides for Raspberry Pi and Armbian: https://forum.openmediavault.org/index.php?thread/28789-installing-omv5-on-raspberry-pi-s-armbian-sbc-s-i386-32-bit-platforms/
