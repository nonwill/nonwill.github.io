Install Google Play Store on Amazon Fire Tablets 🌱

Disclaimer

Remove the microSD card from your Fire tablet before proceeding as it can cause issues during the installation

Enable Unknown Sources

  1. Login into the Fire tablet
  2. Launch Settings
  3. Navigate to Security & Privacy > Apps from Unknown Sources
  4. Select Silk Browser > Enable the Allow from this source option
  5. Close Settings and navigate back to the launch screen

Download and Install APKs

  1. Launch Silk Browser > Navigate to https://bit.ly/apfiretablet | Alternate
  2. Download the following APKs for the target Fire tablet
    • Google Account Manager
    • Google Services Framework
    • Google Play Services
    • Google Play Store
  3. Close Silk Browser and navigate back to the launch screen
  4. Launch the Files application > Navigate to the Downloads folder
  5. Select and install the downloaded com.google.gsf.login apk > Tap Done
  6. Select and install the downloaded com.google.android.gsf apk > Tap Done
  7. Select and install the downloaded com.google.android.gms apk > Tap Done
  8. Select and install the downloaded com.android.vending apk > Tap Done
  9. Once all four APKs have been installed, restart the Fire tablet by holding the power button and selecting Restart
  10. After restarting, log back into the tablet and launch the Play Store app
  11. Login with a Google account
