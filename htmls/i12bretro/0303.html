Quick Start Guide - Setup An Apache/PHP Web Server on Windows in 5 Minutes 🌱

  1. Download XAMPP Download
  3. Install Microsoft Visual C++
  4. Install XAMPP
  5. After the installation completes the XAMPP Control Panel will display
  6. Click the box under Service next to Apache and MySQL to run them as Windows Services
  7. Open a web browser and navigate to http://localhost
  8. Welcome to your fully functional WAMP server in under 5 minutes
