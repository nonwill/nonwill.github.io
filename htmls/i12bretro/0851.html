Install Android 7.1.2 Nougat on the Microsoft Surface RT Tablet 🌱

Disclaimer

This process will run Android 7.1.2 Nougat on the Surface RT tablet from a microSD card. I have tested these steps on my own device (as shown in this video), but I am not responsible if you damage or brick your device.

Things You Will Need

  • A USB flash drive https://amzn.to/3wkR5ju | https://amzn.to/3qkrJ1p | https://amzn.to/3Nhu9b9
  • A microSD card, at least 8 GB https://amzn.to/3JqyxSP

Preparing to Run Android

  1. Insert the microSD card and USB flash drive into a PC
  2. Download Android 7.1.2 Surface RT image Download
  3. Download UEFI Plug and Play Booting Tool for USB Download
  4. Download Balena Etcher Download
  5. Run Balena Etcher
  6. Write the Android 7.1.2 image to the microSD card
  7. Once the write completes, safely remove the microSD card and insert it into the Surface RT's microSD slot
  8. Format the USB flash drive to FAT32

    ⚠ NOTE: This will delete all files on the drive. Make sure to backup any important files before continuing

    1. Right Click Start > File Explorer
    2. Right Click the USB flash drive > Format...
    3. Select FAT32 from the File System dropdown
    4. Name the drive EFI
    5. Make sure Quick Format is checked
    6. Click Start
  9. Extract the downloaded UEFI Plug and Play Booting Tool for USB .zip file to the root of the USB flash drive
  10. Safely remove the USB flash drive

Booting into Android

  1. Power off the Surface RT tablet if it is not already
  2. Connect the prepared USB flash drive to the Surface RT tablet
  3. Hold Vol- and press the power button, releasing the Vol- when Surface logo appears
  4. Once the menu appears, select Env USB option
  5. The Surface RT should begin booting into Android
