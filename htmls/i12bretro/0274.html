Install phpSysInfo on Windows 🌱

  1. Download XAMPP Download
  2. Download phpSysInfo Download
  4. Install Microsoft Visual C++
  5. Right click the downloaded XAMPP file > 7-Zip > Extract here...
  6. Right click the downloaded phpSysInfo file > 7-Zip > Extract here...
  7. Rename the extracted folder phpsysinfo
  8. Make a copy of phpsysinfo/phpsysinfo.ini.new and name it phpsysinfo.ini
  9. Cut the phpsysinfo folder inside the XAMPP/htdocs directory
  10. Copy the extracted XAMPP directory to a safe location to run from, C:\Program Files for example
  11. Run XAMPP/setup_xampp.bat to update the configuration files with the new server location
  12. Navigate to XAMPP/php/ and edit php.ini in a text editor
  13. Search for extension=
  14. Add the following line below one of the existing extensions

    extension=com_dotnet

  15. Save the php.ini file
  16. Navigate to /XAMPP and run xampp-control.exe
  17. Click the Start button next to Apache
  18. Open a web browser and navigate to http://DNSorIP/phpsysinfo
