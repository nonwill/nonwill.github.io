Make Armbian/XFCE Look Like MacOS [XFCE Theming] 🌱

  1. Download the Os Catalina theme Download
  2. Download the Os Catalina icons Download
  3. Extract and install the downloaded archives with the following command:
    # install prerequisites
    sudo apt install plank gtk2-engines-murrine gtk2-engines-pixbuf -y
    # download desktop background
    sudo wget -O /usr/share/backgrounds/xfce/macOS-Catalina-Light-mode.jpg https://cdn.osxdaily.com/wp-content/uploads/2019/06/macOS-Catalina-Light-mode.jpg
    # make sure .icons and .themes exist
    mkdir ~/.themes && mkdir ~/.icons
    # extract the downloaded icons
    sudo tar -xf ~/Downloads/Os-Catalina-Icons*tar.xz -C ~/.icons
    # extract the downloaded theme
    sudo tar -xf ~/Downloads/Os-Catalina-XFCE*tar.xz -C ~/.themes
    # move the plank theme
    mkdir ~/.local/share/plank/themes/Os-Catalina -p
    cp ~/.themes/Os-Catalina-XFCE-4.16/plank/dock.theme ~/.local/share/plank/themes/Os-Catalina/dock.theme
  4. Click the XFCE button > Settings > Desktop
  5. On the Background tab, select the macOS Catalina background
  6. Click the XFCE button > Settings > Window Manager
  7. On the Style tab, select Os Catalina XFCE
  8. Click the XFCE button > Settings > Appearance
  9. On the Style tab, select Os Catalina XFCE
  10. On the Icons tab, select Os Catalina
  11. Click the XFCE button > Settings > Panel
  12. On the Display tab, slide the Row size to taste
  13. Select the Items tab
  14. Select the Whisker menu > Click the Edit icon
  15. Click the Panel Button tab > Click the Icon to edit it
  16. Select All Icons from the drop down
  17. Search logo > Select the apple icon > Click OK
  18. Press ALT+F2 > Type plank
  19. Hold CTRL and right click on the Plank dock > Preferences
  20. Select the Os-Catalina theme
  21. Click the XFCE button > Settings > Session and Startup
  22. On the Application Autostart tab, Click the +Add button
  23. Enter the name Plank and command plank > Click OK
