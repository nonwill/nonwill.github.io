RockStor NAS Install Guide Using Virtualbox 🌱

  1. Download RockStor Download
  2. Launch Virtualbox
  3. Create a New VM by selecting Machine > New

    Name: RockStor
    Machine Folder: C:\VMs
    Type: Linux
    Version: Fedora (64-bit)
    Memory Size: 4096 MB
    Hard disk: Create a virtual hard disk now

  4. Click Create
  5. On the Create Virtual Hard Disk dialog

    Name the virtual disk image RockStor.vdi
    File size: 8 GB
    Hard disk file type: VDI
    Storage on physical hard disk: Dynamically Allocated

  6. Click Create
  7. Select the VM and Click Settings
  8. Select Display
  9. Slide the Video Memory to 128 MB
  10. Select Network
  11. Set the attached to dropdown to Bridged Adapter
  12. Select Storage
  13. Click on the CD-ROM drive
  14. Select the disc dropdown to the right > Choose a virtual optical disc file...
  15. Browse to and select the downloaded RockStor .iso file
  16. Click OK
  17. Make sure the RockStor VM is selected and click Start > Normal
  18. The installation should begin
  19. Click the ROOT PASSWORD option > Enter and confirm the root password > Click Done
  20. After the installation completes select Devices > Optical Drives > Remove disk from virtual drive
  21. Click Force Unmount
  22. Click the Reboot button
  23. Once RockStor has finished booting, login with the username root and the password set during installation
  24. Type ip addr to output the VM network information
  25. Note the IP address
  26. Open a web browser and navigate to https://IPAddress
  27. At the RockStor Setup page, enter a hostname and create an admin username and password > Click Submit
  28. Welcome to RockStor
