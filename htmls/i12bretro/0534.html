Install Homebrew Channel and Apps on Dolphin Emulator 🌱

Preparing the Virtual SD Card

  1. Download 7-zip Download
  2. Download Virtual SD Card Maker Download
  3. Download ImDisk Virtual Disk Driver Download
  4. Install 7-zip by running the .exe
  5. Extract the downloaded Virtual SD Card Maker .zip
  6. Install the ImDisk Virtual Disk Driver
  7. Navigate to the extracted Virtual SD Card Maker /Windows directory
  8. Hold the SHIFT key and right click in the white space > Open PowerShell window here...
  9. Run the following command to create a new blank .raw SD card
    # change the size as needed, 32GB (32768M) is the maximum usable size by Dolphin
    .\mksdcard 4096M .\sd.raw

Installing Homebrew Applications

  1. Back in File Explorer, right click the newly created sd.raw > Mount as ImDisk Virtual Disk
  2. In the ImDisk Virtual Disk dialog, set the drive letter and check the Removable media box > Click OK
  3. Open the newly mounted drive in File Explorer
  4. Create a folder called apps
  5. Copy Wii homebrew applications inside the apps folder, in this example WiiXplorer Download
  6. Once completed copying homebrew applications, right click the mounted virtual SD drive > Eject

Using Homebrew Applications in Dolphin

  1. Download the Homebrew Channel Download
  2. Download the Dolphin emulator Download
  3. Extract the downloaded Dolphin .7z file
  4. Inside the extracted Dolphin directory, create a new file named portable.txt
  5. Launch Dolphin.exe
  6. Copy/cut the sd.raw created earlier to the extracted Dolphin /User/Wii directory
  7. Back in Dolphin, select Tools > Perform Online System Update > United States
  8. After the update completes, select Tools > Install WAD...
  9. Navigate and select the downloaded Homebrew Channel .wad file
  10. Select Tools > Load Wii System Menu
  11. Completed the initial console setup
  12. The familiar Wii launcher should display with Homebrew Channel installed
  13. Select Homebrew Channel to see the homebrew apps installed on the virtual SD card
