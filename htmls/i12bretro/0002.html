Backing Up DD-WRT Settings Using HTTrack 🌱

Disclaimer: This does not backup 100% of the settings in DD-WRT, but it is a great way to capture the settings prior to upgrading so values can be copied/pasted back into place

  1. Download HTTrack Download
  2. Extract the download hhtrack-noinst zip archive
  3. Create a directory to store your DD-WRT backup
  4. Launch WinHTTrack.exe
  5. Click Next
  6. Set Project name: DD-WRT
  7. Set Base path to the directory created in step 3
  8. Click Next
  9. Click Add URL...

    Set URL: to your DD-WRT IP Address
    Login: your DD-WRT user name
    Password: your DD-WRT password

  10. Click OK
  11. Click Next
  12. Wait for the process to complete
  13. Browse the site created by HTTrack
