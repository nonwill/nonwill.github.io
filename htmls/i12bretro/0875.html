Can Proxmox VE Run a Windows XP VM? 🌱

  1. Download a Windows XP x64 .iso Download
  2. Upload the Windows XP x64 .iso file to the Proxmox ISO library
  3. Log into the ProxMox web UI
  4. Right click the ProxMox node name > Create VM
  5. Give the VM a unique ID and Name > Next
  6. Set the Type to Microsoft Windows, Version to XP/2003 and select the Windows XP iso
  7. Leave the defaults on the System tab > Next
  8. On the Hard Disk tab, set the Bus/Device dropdown to IDE and the Disk size to 4 GB or more > Next
  9. On the CPU tab, set Cores to 1 or more > Next
  10. On the Memory tab, set the Memory to 1024 or more > Next
  11. On the Network tab, set the Model field to Realtek RTL8139 > Next
  12. Verify the summary and click Finish
  13. Right click the Windows XP VM in the left navigation pane > Start
  14. Click console in the left sub-navigation menu
  15. Windows Setup will start automatically
  16. Press Enter to Continue with Setup
  17. Press F8 to Agree to the Licensing Agreement
  18. Press Enter to Install to the Unpartitioned space
  19. Select the top option to Format the partition using the NTFS file system (Quick) > Press Enter
  20. Wait for Windows to reboot the VM
  21. Do not press anything to boot the CD-ROM
  22. Wait for Windows to install and configure drivers and devices
  23. Click Next
  24. Enter a name and organization, click Next
  25. Enter a valid Windows XP product key (DG8FV-B9TKY-FRT9J-6CRCC-XPQ4G), click Next
  26. Give the Computer a name
  27. Set an administrator password and re-type to confirm > Click Next
  28. Set the date/time and select a timezone > Click Next
  29. Click the Next button to accept default and continue installing Windows XP
  30. Click OK to allow Windows to adjust the screen resolution
  31. Click OK if the screen resolution looks better
  32. Click Next at the Welcome Screen
  33. Select Not right now for automatic updates > Click Next
  34. Click Skip on the internet connection screen
  35. Select No, not at this time on the Register with Microsoft screen > Click Next
  36. Enter a user name > Click Next
  37. Click Finish
  38. Welcome back to Windows XP
