Transform FireTV Stick Into An Awesome Emulation Device, Part 2 🌱

Setting Up FTP Access to USB Drive

  1. On the FireTV device search for ES File Explorer
  2. Install ES File Explorer
  3. Launch ES File Explorer
  4. Scroll down the left navigation to the USB flash drive
  5. Navigate in the right pane to any file > Long press the file > Select Copy
  6. Navigate back to the root of the USB flash drive > Select Paste
  7. When prompted, allow ES File Explorer access to the USB flash drive
  8. Long press the copied file > Select Delete > Confirm deleting the file
  9. Scroll up on left navigation and select Network > View on PC
  10. Click the Settings button at the bottom
  11. Click the Mount Path option and set the path to the root of the USB flash drive
  12. Click the big Turn On button to start the FTP server
  13. The IP address and port will display, by default anonymous access is enabled
  14. Download FileZilla FTP client Download
  15. Extract the downloaded .zip file
  16. Launch FileZilla.exe
  17. Enter the IP address and port (3721 by default) in the quick connect fields and press Enter to connect
  18. If everything is setup correctly, a listing of folders on the USB flash drive should be displayed in the right pane

Customizing Artwork

With FTP access setup, artwork can be easily customized

NOTE: Image files must match the ROM file name exactly, IE artwork for "i12bretro Strikes Back (USA).zip" must be name "i12bretro Strikes Back (USA).png"

  1. Download the artwork to be used (EmuMovies and LaunchBox forums are great resources)
  2. Launch FileZilla
  3. Enter the IP address and port (3721 by default) in the quick connect fields and press Enter to connect
  4. Navigate the USB flash drive in the right pane to the Covers or Screenshots directory > Find the folder for the system to modify
  5. Drag/Drop files to copy into the right pane
  6. Launch Dig and the updated artwork should appear, if it does not, back out of the system and reload it again 

Installing Themes

Some of my favorite themes are Blast 16, Comic Book, Mega-PI USA, Unified Arkade, NES, Retroman UltimateVideo, Riivive, SNES Classic Official UI

NOTE: Not all themes support every system

  1. Download the theme(s) to be installed Download
  2. Launch FileZilla.exe
  3. Enter the IP address and port (3721 by default) in the quick connect fields and press Enter to connect
  4. Drag/Drop downloaded theme zip files into the right pane
  5. Launch Dig and go to > Options > Themes > Install theme
  6. Navigate to the root of the USB flash drive and select the theme .zip file to install 
  7. Click Confirm
  8. Dig will extract the contents of the zip file and install the theme
  9. To set the theme go to Options > Themes > Select theme and select it from the dropdown
