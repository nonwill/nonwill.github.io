Install eDEX-UI on Debian - Awesome Terminal Emulator 🌱

  1. Download eDEX-UI Download
  2. Launch a terminal and execute the following command to make the .AppImage file executable
    cd ~/Downloads
    chmod a+x eDEX-UI*.AppImage
    ./eDEX-UI*.AppImage
  3. Nerd out over this awesome terminal emulator

Source: https://github.com/GitSquared/edex-ui
