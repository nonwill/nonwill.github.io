Downloads and Flashing to MicroSD

  1. Download Batocera Download
  2. Download Balena Etcher Download
  3. Run Balena Etcher
  4. Write the Batocera image to the USB flash drive

Booting to Batocera for the First Time

  1. Power off the PC
  2. Plug in the Batocera USB flash drive
  3. Power on the host computer, this time choosing to boot to the USB flash drive
    The steps required to do this will vary based on motherboard, typically pressing F2, F12, DEL or ESC during boot will bring up a menu to set the boot device
  4. You should see the Batocera splash screen
  5. Setup Your Controller
  6. Game
