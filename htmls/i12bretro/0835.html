Updating to Windows 11 22H2 on Unsupported Hardware [No TPM | No Secure Boot] 🌱

  1. Download the Windows 11 TPM/SecureBoot bypass script Download
  2. Download the Windows 11 22H2 Installation .iso file Download
  3. Extract the downloaded MediaCreationTool .zip archive
  4. Navigate into the downloaded folder /bypass11 > Right click on Skip_TPM_Check_on_Dynamic_Update.cmd > Run as administrator
  5. Press Enter once the script completes to close the command prompt
  6. Double click the downloaded Windows 11 22H2 .iso file to mount it
  7. In File Explorer, navigate to the mounted Windows 11 22H2 disc image and double click Setup.exe
  8. Once the installer loads, click the Change how Setup downloads updates option > Select Not right now > Click Next
  9. Step through the rest of the setup as normal
  10. After a few reboots the system will be updated to Windows 11 22H2

Cleanup After Testing

After verifying the update has not caused any issues, follow the steps below to free up the space used by the Windows.old backup

  1. Right click on the Start button > Settings
  2. Select Storage > Temporary files
  3. Check the box next to Previous Windows installation(s) > Click Remove files
