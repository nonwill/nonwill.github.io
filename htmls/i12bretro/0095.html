Easy to Use Bootable Flash Drive with Ventoy 🌱

What is Ventoy?

Ventoy is an open source tool to create bootable USB drive for ISO/WIM/IMG/VHD(x)/EFI files. With ventoy, you don't need to format the disk over and over, you just need to copy the ISO/WIM/IMG/VHD(x)/EFI files to the USB drive and boot them directly. You can copy many files at a time and ventoy will give you a boot menu to select them -https://www.ventoy.net/

Things You Will Need

  • A USB flash drive, at least 32 GB of free space https://amzn.to/3qkrJ1p | https://amzn.to/3Nhu9b9
  1. Download the Ventoy installer Download
  2. Extract the downloaded .zip file
  3. Run Ventoy2Disk.exe
  4. Plug in a USB flash drive at least 32 GB in size
  5. Click the refresh icon
  6. Select the flash drive from the device dropdown
  7. Click the Install button
  8. After the installation completes, simply copy .iso files to the Ventoy partition
  9. Safely remove the USB flash drive
  10. Boot to the flash drive when needed and the Ventoy boot menu will appear listing the .iso files
