Awesome Terminal Emulator Running on Armbian S905x/S912/H6 TV Box 🌱

  1. Download eDEX-UI Download
  2. Launch a terminal and execute the following command to download dependencies and make the .AppImage file executable
    # enable armv7 / 32-bit software
    sudo dpkg --add-architecture armhf
    # install required armv7 / 32-bit packages
    sudo apt install libc6:armhf libstdc++6:armhf libfuse2:armhf zlib1g-dev:armhf libglib2.0-dev:armhf libnss3:armhf libgtk-3-0:armhf libx11-xcb1:armhf libxtst6:armhf libxss-dev:armhf libasound2:armhf
    # launch eDEX-UI
    cd ~/Downloads
    chmod a+x eDEX-UI*.AppImage
    ./eDEX-UI*.AppImage
  3. Nerd out over this awesome terminal emulator
