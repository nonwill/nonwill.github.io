Add OpenFire Instant Messaging in Mozilla Thunderbird 🌱

  1. Launch Mozilla Thunderbird
  2. Select File > New > Chat Account...
  3. Select XMPP > Click Next
  4. Input the OpenFire username and domain > Click Next
  5. Input the chat user passwords > Click Next
  6. Set the Local Alias field
  7. From the Connection security dropdown, select Require encryption is SSL has been setup, otherwise select Use encryption if available
  8. In the Server field enter the IP address or DNS name for the OpenFire server
  9. Click Next
  10. Click Finish
  11. Enjoy instant messaging inside the Thunderbird e-mail client
